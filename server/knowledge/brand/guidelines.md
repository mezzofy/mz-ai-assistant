# Mezzofy Brand Guidelines

## Colors
- Primary Orange: #f97316
- Black: #000000
- White: #ffffff
- Light Orange (alternate rows / backgrounds): #fef3ea

## Typography
- Primary Font: Inter (sans-serif); fallback: Helvetica Neue, Arial
- Heading Weight: 700 (Bold)
- Body Weight: 400 (Regular)
- Monospace Font: JetBrains Mono (code/data); fallback: Courier New

## Voice & Tone
- Confident and authoritative — state facts directly, avoid hedging language
- Friendly and approachable — frame interactions as partnerships, not transactions
- Results-focused — always tie features and data back to business outcomes
- Clear and concise — no jargon, no passive voice, no filler phrases
- Positive but honest — do not oversell; acknowledge limitations plainly

## Document Standards

### All Documents (apply to every format)
- Always include "Generated by Mezzofy AI Assistant" in footer
- Use orange (#f97316) for all primary headings (H1, H2, slide titles)
- Use black (#000000) for body text
- Use white (#ffffff) backgrounds for content areas
- Company name is always "Mezzofy" — never "mezzofy" or "MEZZOFY"
- Date format: DD Month YYYY (e.g., 21 March 2026)

### PowerPoint / PPTX Presentations
- Cover slide: Black background, orange accent bars, white title text
- Content slides: White background, orange top bar (6pt), orange section headings
- Divider / section break slides: Orange background, white title
- Thank You / closing slide: Black background, orange logo area, white text
- Minimum structure: Cover → Agenda → [Content Slides] → Thank You
- Font sizes: Title 36–44pt, Subtitle 24–28pt, Body 18–20pt, Caption 14pt
- Maximum 6 bullet points per slide; prefer visuals over walls of text

### PDF Reports
- Page header: "Mezzofy" branding top-right in orange on every page
- Page footer: "Generated by Mezzofy AI Assistant | Page N" centered
- Table headers: Orange background (#f97316), white text, bold
- Alternate table rows: Light orange (#fef3ea)
- Charts/graphs: Use orange as primary series color, black for secondary
- Section headings (H1): Orange, 16pt bold; H2: Black, 13pt bold

### Word Documents / DOCX
- Document header: "Mezzofy AI Assistant" right-aligned, orange, 10pt
- Document footer: "Generated by Mezzofy AI Assistant" centered, 9pt grey
- H1: Orange (#f97316), 20pt bold
- H2: Orange (#f97316), 16pt bold
- H3: Black, 13pt bold, slight indent
- Body: Black, 11pt, 1.15 line spacing
- Tables: Orange header row, alternating light orange body rows

### Excel / XLSX Spreadsheets
- Header row: Orange background (#f97316), white bold text
- Alternate data rows: Light orange (#fef3ea) and white
- Total / summary rows: Black background, white bold text
- Column headers: Bold, centered
- Numeric data: Right-aligned; text data: Left-aligned
- Tab/sheet names: Title case, descriptive (e.g., "Monthly Sales", not "Sheet1")

## Logo Usage
- Minimum height: 24px / 18pt
- Use on white or black backgrounds only — never on coloured backgrounds
- Do not alter proportions, apply effects, or change the orange colour
- Clear space around logo: minimum 1× logo height on all sides

## What to Avoid
- Do not use gradients — solid colours only
- Do not use more than 3 typeface weights in a single document
- Do not use red or green as accent colours (reserved for status indicators)
- Do not include clipart or low-resolution images
- Do not use Comic Sans, Times New Roman, or decorative fonts

### Placement Guidelines
- **PDF**: Top-right of page header, height 32px, beside document type label
- **PPTX Cover slide**: Bottom-left corner, height 40px, on black background → use light logo
- **PPTX Content slides**: Top-right of orange header bar, height 24px → use light logo (on orange)
- **PPTX Thank You slide**: Centered, height 60px, on black background → use light logo
- **DOCX**: Right-aligned in page header section, height 28px → use dark logo
