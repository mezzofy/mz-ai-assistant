# Mezzofy Brand Guidelines
## AI Generation Reference — PPTX / DOCX / XLSX / PDF

> **Purpose:** This document is the authoritative reference for the Mezzofy AI Assistant when generating branded documents. Every measurement, colour value, font size, and layout rule below is derived directly from the four QA-passed template files (`mezzofy_template.pptx`, `mezzofy_template.docx`, `mezzofy_template.xlsx`, `mezzofy_template.pdf`). Do not deviate from these values.

---

## 1. Brand Tokens

### Colours
| Token | Hex | ARGB (openpyxl) | Usage |
|---|---|---|---|
| Primary Orange | `#F97316` | `FFF97316` | Headings, table headers, accent bars, borders, KPI stats |
| Black | `#000000` | `FF000000` | Body text, XLSX total row background |
| White | `#FFFFFF` | `FFFFFFFF` | Content backgrounds, text on dark/orange |
| Light Orange | `#FEF3EA` | `FFFEF3EA` | Alternate table rows, callout card backgrounds |
| Grey (footer/meta) | `#888888` | — | Footer text, captions, metadata labels |
| Grey (light) | `#AAAAAA` | — | PPTX footer text on white slides |
| Dark Grey (sub-text) | `#555555` | — | KPI card supporting text (PPTX only) |

### Typography
| Format | Font | Fallback |
|---|---|---|
| PPTX | `Inter` (via pptxgenjs `fontFace`) | System sans-serif |
| DOCX | `Arial` (via docx.js `font`) | Helvetica Neue |
| XLSX | `Arial` (via openpyxl `Font(name=...)`) | Helvetica Neue |
| PDF | `Helvetica` / `Helvetica-Bold` (reportlab built-in) | — |

### Voice & Tone
- **Confident** — state facts directly, no hedging language
- **Results-focused** — tie every feature or data point to a business outcome
- **Clear and concise** — no jargon, no passive voice, no filler phrases
- **Positive but honest** — do not oversell; acknowledge limitations plainly
- Company name is always **"Mezzofy"** — never "mezzofy" or "MEZZOFY"
- Date format: **DD Month YYYY** (e.g., 21 March 2026)

### pptxgenjs — Core Helper Functions
```javascript
const pptxgen = require("pptxgenjs");
const prs = new pptxgen();

// Brand tokens
const ORANGE       = "F97316";
const BLACK        = "000000";
const WHITE        = "FFFFFF";
const LIGHT_ORANGE = "FEF3EA";
const GREY         = "888888";
const FONT         = "Inter";

// Must call on every content slide (Slides 2, 4, 5, 6)
function addTopBar(slide) {
  slide.addShape(prs.ShapeType.rect, {
    x: 0, y: 0, w: "100%", h: 0.08,
    fill: { color: ORANGE }, line: { type: "none" },
  });
}

// Eyebrow section label (e.g. "AGENDA", "KEY METRICS")
function addEyebrow(slide, text, x = 0.5, y = 0.20) {
  slide.addText(text.toUpperCase(), {
    x, y, w: 9, h: 0.25,
    fontSize: 9, bold: true, color: ORANGE,
    fontFace: FONT, charSpacing: 2,
  });
}

// Slide title (orange, 36pt bold)
function addTitle(slide, text) {
  slide.addText(text, {
    x: 0.5, y: 0.55, w: 12, h: 0.55,
    fontSize: 36, bold: true, color: ORANGE, fontFace: FONT,
  });
}

// Footer on white/light slides
function addFooter(slide, color = "AAAAAA") {
  slide.addText("Generated by Mezzofy AI Assistant", {
    x: 0, y: 7.25, w: "100%", h: 0.22,
    fontSize: 9, color, fontFace: FONT, align: "center",
  });
}
```

### docx.js — Core Scaffold
```javascript
const {
  Document, Packer, Paragraph, TextRun, Table, TableRow, TableCell,
  Header, Footer, AlignmentType, HeadingLevel, LevelFormat, BorderStyle,
  WidthType, ShadingType, VerticalAlign, PageNumber,
  TabStopType, TabStopPosition,
} = require("docx");

// Header: right-aligned "Mezzofy AI Assistant" in orange, rule below
function makeHeader() {
  return new Header({
    children: [new Paragraph({
      border: { bottom: { style: BorderStyle.SINGLE, size: 6, color: "F97316", space: 1 } },
      tabStops: [{ type: TabStopType.RIGHT, position: TabStopPosition.MAX }],
      spacing: { after: 60 },
      children: [
        new TextRun({ text: "", size: 20, font: "Arial" }),
        new TextRun({ text: "\t", size: 20, font: "Arial" }),
        new TextRun({ text: "Mezzofy AI Assistant", size: 20, font: "Arial",
                      color: "F97316", bold: true }),
      ],
    })],
  });
}

// Footer: centered "Generated by Mezzofy AI Assistant  |  Page N"
function makeFooter() {
  return new Footer({
    children: [new Paragraph({
      alignment: AlignmentType.CENTER,
      border: { top: { style: BorderStyle.SINGLE, size: 4, color: "EEEEEE", space: 1 } },
      spacing: { before: 60 },
      children: [
        new TextRun({ text: "Generated by Mezzofy AI Assistant  |  Page ",
                      size: 18, font: "Arial", color: "888888" }),
        new TextRun({ children: [PageNumber.CURRENT], size: 18,
                      font: "Arial", color: "888888" }),
      ],
    })],
  });
}

// Branded table (3-column example — adjust columnWidths to match column count)
function makeTable(headers, rows) {
  const colWidths = [4680, 2340, 2340]; // must sum to 9360
  const border = { style: BorderStyle.SINGLE, size: 6, color: "DDDDDD" };
  const borders = { top: border, bottom: border, left: border, right: border };
  const cellMargins = { top: 80, bottom: 80, left: 140, right: 140 };

  const headerRow = new TableRow({
    tableHeader: true,
    children: headers.map((h, i) => new TableCell({
      borders, width: { size: colWidths[i], type: WidthType.DXA }, margins: cellMargins,
      shading: { fill: "F97316", type: ShadingType.CLEAR },
      verticalAlign: VerticalAlign.CENTER,
      children: [new Paragraph({ children: [
        new TextRun({ text: h, bold: true, color: "FFFFFF", size: 22, font: "Arial" })
      ]})],
    })),
  });

  const dataRows = rows.map((row, ri) => new TableRow({
    children: row.map((cell, ci) => new TableCell({
      borders, width: { size: colWidths[ci], type: WidthType.DXA }, margins: cellMargins,
      shading: { fill: ri % 2 === 0 ? "FFFFFF" : "FEF3EA", type: ShadingType.CLEAR },
      verticalAlign: VerticalAlign.CENTER,
      children: [new Paragraph({ children: [
        new TextRun({ text: cell, size: 22, font: "Arial", color: "000000" })
      ]})],
    })),
  }));

  return new Table({
    width: { size: 9360, type: WidthType.DXA },
    columnWidths: colWidths,
    rows: [headerRow, ...dataRows],
  });
}
```


### openpyxl — Core Style Functions
```python
from openpyxl.styles import Font, PatternFill, Alignment, Border, Side

ORANGE       = "F97316"
WHITE        = "FFFFFF"
LIGHT_ORANGE = "FEF3EA"
BLACK        = "000000"
FONT_NAME    = "Arial"

def fill(hex_color):
    return PatternFill("solid", fgColor=hex_color)

def thin_border(color="DDDDDD"):
    s = Side(style="thin", color=color)
    return Border(left=s, right=s, top=s, bottom=s)

def style_header(ws, row, col_count):
    for c in range(1, col_count + 1):
        cell = ws.cell(row=row, column=c)
        cell.fill = fill(ORANGE)
        cell.font = Font(name=FONT_NAME, bold=True, color=WHITE, size=10)
        cell.alignment = Alignment(horizontal="center", vertical="center")
        cell.border = thin_border(ORANGE)

def style_data_row(ws, row, col_count, alt=False):
    # alt=True  → even rows  → light orange (#FEF3EA)
    # alt=False → odd rows   → white (#FFFFFF)
    for c in range(1, col_count + 1):
        cell = ws.cell(row=row, column=c)
        cell.fill = fill(LIGHT_ORANGE if alt else WHITE)
        cell.font = Font(name=FONT_NAME, color=BLACK, size=10)
        cell.alignment = Alignment(horizontal="left", vertical="center")
        cell.border = thin_border()

def style_total_row(ws, row, col_count):
    for c in range(1, col_count + 1):
        cell = ws.cell(row=row, column=c)
        cell.fill = fill(BLACK)          # ARGB stored as FF000000 — correct
        cell.font = Font(name=FONT_NAME, bold=True, color=WHITE, size=10)
        cell.alignment = Alignment(horizontal="right", vertical="center")
        cell.border = thin_border(BLACK)
```

### Header/Footer Callback
```python
from reportlab.lib.units import inch
from reportlab.lib.colors import HexColor

ORANGE = HexColor("#F97316")
GREY   = HexColor("#888888")

def draw_header_footer(canvas, doc):
    canvas.saveState()
    W, H = doc.pagesize

    # Orange top rule
    canvas.setFillColor(ORANGE)
    canvas.rect(0, H - 0.55*inch, W, 0.04*inch, fill=1, stroke=0)

    # "Mezzofy" top-right in orange
    canvas.setFont("Helvetica-Bold", 9)
    canvas.setFillColor(ORANGE)
    canvas.drawRightString(W - 0.65*inch, H - 0.45*inch, "Mezzofy")

    # Document type top-left in grey
    canvas.setFont("Helvetica", 8)
    canvas.setFillColor(GREY)
    canvas.drawString(0.65*inch, H - 0.45*inch, doc.title or "Report")

    # Grey rule below header
    canvas.setStrokeColor(HexColor("#EEEEEE"))
    canvas.setLineWidth(0.5)
    canvas.line(0.65*inch, H - 0.62*inch, W - 0.65*inch, H - 0.62*inch)

    # Grey rule above footer
    canvas.line(0.65*inch, 0.58*inch, W - 0.65*inch, 0.58*inch)

    # Footer text centred
    canvas.setFont("Helvetica", 8)
    canvas.setFillColor(GREY)
    canvas.drawCentredString(W/2, 0.38*inch,
        f"Generated by Mezzofy AI Assistant  |  Page {doc.page}")

    canvas.restoreState()

# Build call:
doc.build(story, onFirstPage=draw_header_footer, onLaterPages=draw_header_footer)
```

### Critical reportlab Rules
- **Never use Unicode subscripts/superscripts** (`₀₁²³⁰¹`) — renders as solid black boxes in Helvetica
  - Use `<sub>` and `<super>` XML tags inside `Paragraph()` instead
- **Always `repeatRows=1`** on `Table()` — header must repeat across page breaks
- **Column widths from content width** — always derive from `W - left_margin - right_margin`, never hardcode

## 8. What to Avoid

| Rule | Applies to |
|---|---|
| No gradients — solid colours only | All |
| No more than 3 typeface weights in a single document | All |
| No red or green accent colours (reserved for XLSX financial indicators) | PPTX, DOCX, PDF |
| No accent underlines beneath slide titles | PPTX |
| No `WidthType.PERCENTAGE` on tables | DOCX |
| No `ShadingType.SOLID` on table cells | DOCX |
| No Unicode bullets `•` as raw text — use `LevelFormat.BULLET` | DOCX |
| No Unicode subscripts/superscripts — use `<sub>`/`<super>` tags | PDF |
| No hardcoded Python-calculated values in cells — use Excel formulas | XLSX |
| No `data_only=True` when saving a workbook — destroys all formulas | XLSX |
| No Comic Sans, Times New Roman, or decorative fonts | All |
| No low-resolution images or clipart | All |
