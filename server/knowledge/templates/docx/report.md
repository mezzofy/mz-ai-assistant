# Template: Business Report (DOCX)

## Purpose
Formal business reports: market analysis, feasibility studies, operational reviews, investigation findings.

## Document Structure

1. **Cover Page** — Report title, author(s), department, date, version, classification (Internal / Confidential).
2. **Table of Contents** — Auto-generated; list all sections with page numbers.
3. **Executive Summary** — 1–2 page summary of purpose, key findings, and recommendations. Stand-alone readable.
4. **Introduction / Background** — Context, scope, objectives, methodology, limitations.
5. **Findings / Analysis** — Core content section. Use H2 subsections per topic area. Include data, charts, evidence.
6. **Discussion** — Interpretation of findings; what they mean for the business.
7. **Recommendations** — Numbered list of actionable recommendations, each with owner and priority.
8. **Conclusion** — 1–2 paragraphs summarising the path forward.
9. **References / Sources** — Numbered bibliography or footnotes.
10. **Appendices** — Supporting data, raw tables, interview notes, methodology detail.

## Tone
Professional, objective, evidence-based. Use active voice. Cite sources for all factual claims.

## Formatting Notes
- H1 sections start on a new page
- Tables: caption above (Table N: Description); figures: caption below (Figure N: Description)
- All data sources cited inline [N] and listed in References
