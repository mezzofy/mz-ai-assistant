# Template: Data Report / Analysis (XLSX)

## Purpose
Structured data exports, analysis workbooks, and tabular reports. Used for sales data, customer exports, financial summaries, and ad-hoc analysis.

## Sheet Structure

### Sheet 1: Summary
- Title row (merged, orange background): Report name, period, generated date
- Key metrics table: Metric | Value | vs Prior Period | vs Target
- Notes on data scope and exclusions

### Sheet 2: [Main Data]
- Header row: Orange background, white bold text, auto-filter enabled
- Data rows: Alternating white and light orange (#fef3ea)
- Total row at bottom: Black background, white bold text
- Freeze panes: Row 1 (header) frozen

### Sheet 3: Pivot / Analysis (if applicable)
- Pivot tables derived from Sheet 2 data
- Charts: Orange primary series, black secondary series
- Chart titles: Descriptive (include metric name and period)

### Sheet 4: Metadata
- Generated by: Mezzofy AI Assistant
- Generated at: [datetime]
- Data source: [source description]
- Filter criteria applied: [list]
- Column definitions: Column name | Data type | Description

## Formatting Notes
- Date columns: DD/MM/YYYY format
- Currency: #,##0.00 format with currency symbol in header
- Percentages: 0.0% format
- Column widths: Auto-fit to content, minimum 80px
- No merged cells in data rows (only in title/summary rows)
