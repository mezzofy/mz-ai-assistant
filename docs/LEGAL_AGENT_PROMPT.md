## CONTEXT — What We Are Building

We are adding **Leo (Legal Agent)** — a new specialist AI team member to the Mezzofy AI Assistant.

Leo sits alongside the existing specialist agents (Research, Developer, Scheduler, Finance, Sales, Marketing, Support, HR, Management). Like all agents, Leo is a **first-class entity** — an autonomous AI team member with its own DB row, skill set, RAG knowledge namespace, and persona. Leo is NOT a user. Leo is never authenticated via JWT.

**Leo's mandate:**
- Review uploaded legal documents (contracts, NDAs, MOU, shareholder agreements, term sheets, employment contracts, vendor agreements)
- Draft and generate business contracts and legal agreements from templates
- Provide legal advisory notes on international business matters
- Apply jurisdiction-specific knowledge for: **Singapore, Hong Kong, Malaysia, Middle East (UAE/Saudi Arabia/Qatar), and Cayman Islands**
- Flag legal risk areas, missing clauses, and compliance gaps
- Work cross-functionally — can be delegated to by Management Agent, or invoked directly by authorised users

**Important legal disclaimer requirement:** Every Leo output must include a standard disclaimer that this is AI-generated legal analysis for reference only and does not constitute professional legal advice. Users should consult a qualified solicitor for binding decisions.

## PHASE 1 — Legal Agent Python Class

### New file: `server/agents/legal_agent.py`

```python
"""
Legal Agent — Leo
Specialist in international business law, document review, contract generation,
and jurisdiction-specific legal advisory for Singapore, Hong Kong, Malaysia,
Middle East, and Cayman Islands.
"""

from .base_agent import BaseAgent

LEGAL_DISCLAIMER = (
    "\n\n---\n"
    "⚖️ **Legal Disclaimer:** This analysis is generated by an AI assistant "
    "for informational and reference purposes only. It does not constitute "
    "professional legal advice and should not be relied upon as such. "
    "For binding legal decisions, please consult a qualified solicitor or "
    "legal practitioner admitted in the relevant jurisdiction."
)

LEGAL_TRIGGERS = [
    # Document types
    "contract", "agreement", "nda", "non-disclosure", "mou", "memorandum of understanding",
    "term sheet", "shareholder agreement", "employment contract", "vendor agreement",
    "service agreement", "licensing agreement", "partnership agreement", "joint venture",
    "loan agreement", "ip assignment", "consultancy agreement", "distribution agreement",
    "franchise agreement", "lease agreement", "letter of intent", "loi",
    # Actions
    "review contract", "review agreement", "review document", "draft contract",
    "draft agreement", "generate contract", "legal review", "legal advice",
    "legal opinion", "legal check", "compliance check", "risk assessment",
    "redline", "mark up", "legal analysis", "clause", "indemnity", "liability",
    "governing law", "jurisdiction", "arbitration", "dispute resolution",
    "intellectual property", "ip rights", "data protection", "gdpr", "pdpa",
    # Jurisdictions
    "singapore law", "hong kong law", "malaysia law", "cayman islands",
    "uae law", "saudi law", "middle east law", "english law",
    "companies act", "employment act", "pdpa", "contract act",
    # Legal concepts
    "force majeure", "liquidated damages", "warranties", "representations",
    "termination clause", "non-compete", "confidentiality", "exclusivity",
    "intellectual property", "assignment", "novation", "indemnification",
]


class LegalAgent(BaseAgent):
    """
    Leo — Legal Agent
    Handles document review, contract drafting, and legal advisory.
    Jurisdiction coverage: SG, HK, MY, UAE/SA/QA, Cayman Islands.
    """

    def can_handle(self, task: dict) -> bool:
        """Return True if message contains legal keywords or is explicitly routed here."""
        if task.get("agent_override") == "legal":
            return True
        message = task.get("message", "").lower()
        return any(trigger in message for trigger in LEGAL_TRIGGERS)

    async def execute(self, task: dict) -> dict:
        """
        Route to the appropriate legal workflow based on task type.

        Detects task type via LLM classification:
          - "document_review"     → review_document()
          - "contract_generation" → generate_contract()
          - "legal_advisory"      → provide_legal_advice()
          - "clause_extraction"   → extract_clauses()
          - "risk_assessment"     → assess_legal_risk()
        """
        task_id = await self.log_task_start(task)

        try:
            # Classify the legal task type
            task_type = await self._classify_legal_task(task)
            task["legal_task_type"] = task_type

            if task_type == "document_review":
                result = await self.review_document(task)
            elif task_type == "contract_generation":
                result = await self.generate_contract(task)
            elif task_type == "clause_extraction":
                result = await self.extract_clauses(task)
            elif task_type == "risk_assessment":
                result = await self.assess_legal_risk(task)
            else:
                # Default: legal advisory / Q&A
                result = await self.provide_legal_advice(task)

            # Append disclaimer to all outputs
            if "content" in result:
                result["content"] += LEGAL_DISCLAIMER
            if "summary" in result:
                result["summary"] += LEGAL_DISCLAIMER

            await self.log_task_complete(task_id, result)
            return result

        except Exception as e:
            await self.log_task_failed(task_id, str(e))
            raise

    # ------------------------------------------------------------------
    # Workflow: Document Review
    # ------------------------------------------------------------------
    async def review_document(self, task: dict) -> dict:
        """
        Review an uploaded legal document (PDF or DOCX).

        Workflow:
        1. _load_skill("document_review") — validates file type, extracts text
        2. _load_knowledge() — load jurisdiction-specific legal knowledge
        3. _classify_jurisdiction(text) — detect governing law from document
        4. LLM: structured analysis prompt (see below)
        5. pdf_generator → branded review report PDF
        6. Return: analysis + PDF artifact

        LLM prompt structure:
          "You are Leo, a specialist legal analyst at Mezzofy.
           Review the following {document_type} under {jurisdiction} law.
           Provide:
           1. Executive Summary (what type of agreement, parties, key dates)
           2. Key Terms & Obligations (for each party)
           3. Risk Flags (clauses that favour the counterparty, missing protections)
           4. Missing Standard Clauses (what should be there but isn't)
           5. Jurisdiction-Specific Notes ({jurisdiction} law considerations)
           6. Recommended Actions (ranked by priority: Critical / High / Medium / Low)

           Document:\n{document_text}"
        """

    async def generate_contract(self, task: dict) -> dict:
        """
        Generate a new business contract or legal agreement.

        Workflow:
        1. _load_skill("contract_drafting")
        2. _load_knowledge() — load relevant templates from knowledge/legal/templates/
        3. LLM: extract contract parameters from user message:
           - Contract type, Parties (names, entity types, jurisdictions)
           - Key commercial terms, Governing law, Special requirements
        4. LLM: draft full contract using template + parameters
        5. docx_ops.create_docx → Word document (primary format)
        6. pdf_ops.create_pdf → PDF version
        7. Return: both artifacts + plain-language summary of key terms

        Contract types supported:
          - Non-Disclosure Agreement (NDA / CNDA)
          - Service Agreement
          - Consultancy Agreement
          - Employment Contract
          - Vendor / Supplier Agreement
          - Memorandum of Understanding (MOU)
          - Letter of Intent (LOI)
          - IP Assignment Agreement
          - Distribution Agreement
          - Shareholders Agreement (basic)
          - Joint Venture Agreement (basic)
        """

    async def extract_clauses(self, task: dict) -> dict:
        """
        Extract and categorise specific clauses from a document.

        Workflow:
        1. document_review skill → extract full text
        2. LLM: identify and extract named clause types
           (e.g. "extract all indemnity clauses, liability caps, and termination provisions")
        3. Return: structured JSON of found clauses + page references + plain summaries
        """

    async def assess_legal_risk(self, task: dict) -> dict:
        """
        Produce a legal risk assessment report for a document or situation.

        Workflow:
        1. document_review skill (if file attached) OR advisory knowledge (if text query)
        2. LLM: generate risk matrix:
           - Risk item | Severity (Critical/High/Medium/Low) | Likelihood |
             Jurisdiction | Recommended Mitigation
        3. pdf_generator → formatted risk matrix report
        4. Return: risk matrix + PDF artifact
        """

    async def provide_legal_advice(self, task: dict) -> dict:
        """
        Answer a legal question or provide advisory guidance.

        Workflow:
        1. _load_knowledge() → load all jurisdiction knowledge files
        2. Detect jurisdiction from query (or ask user to confirm)
        3. LLM: structured advisory response:
           - Direct answer to the question
           - Applicable laws / regulations cited
           - Jurisdiction-specific nuances
           - Practical recommendations
           - When to engage a qualified solicitor
        4. Return: advisory text (no document generated for Q&A)
        """

    # ------------------------------------------------------------------
    # Helpers
    # ------------------------------------------------------------------
    async def _classify_legal_task(self, task: dict) -> str:
        """
        Use LLM to classify the legal task type from message + attachments.
        Returns one of: document_review | contract_generation | clause_extraction |
                        risk_assessment | legal_advisory
        """

    async def _classify_jurisdiction(self, document_text: str) -> str:
        """
        Detect the governing law / jurisdiction from document text.
        Returns: "singapore" | "hong_kong" | "malaysia" | "uae" |
                 "saudi_arabia" | "qatar" | "cayman_islands" | "english" | "unknown"
        """

    def _get_jurisdiction_knowledge_file(self, jurisdiction: str) -> str:
        """Map jurisdiction string to knowledge file path."""
        return f"knowledge/legal/jurisdictions/{jurisdiction}.md"
```

class DocumentReviewSkill:
    """
    Extracts and pre-processes legal document text for LLM analysis.
    Handles PDF (via PyPDF2/pdfplumber) and DOCX (via python-docx).
    """

    async def extract_legal_document(self, file_path: str, file_type: str = "auto") -> dict:
        """
        Extract full text from PDF or DOCX.
        Returns: { text, page_count, word_count, file_type, metadata }
        Raises: UnsupportedFileTypeError if not PDF or DOCX.
        Max document size: 50MB / 200 pages. Truncate with warning if exceeded.
        """

    async def identify_document_type(self, document_text: str) -> dict:
        """
        Use heuristics + LLM to classify document type.
        Returns: {
          contract_type: str,       # "NDA" | "Service Agreement" | "Employment Contract" | ...
          governing_law: str,       # jurisdiction string
          parties: list[str],
          effective_date: str | None,
          expiry_date: str | None,
          is_executed: bool         # signed or draft
        }
        """

    async def detect_parties(self, document_text: str) -> list[dict]:
        """
        Extract party information.
        Returns list of: { name, entity_type, jurisdiction, role }
        role: "disclosing_party" | "receiving_party" | "service_provider" |
              "client" | "employer" | "employee" | "licensor" | "licensee" | etc.
        """

    async def detect_governing_law(self, document_text: str) -> dict:
        """
        Find governing law clause.
        Returns: { jurisdiction, governing_law_text, arbitration_body,
                   dispute_resolution: "arbitration" | "litigation" | "mediation" | "negotiation",
                   arbitration_seat: str | None }
        """

    Generates business contracts using Jinja2 templates stored in

    """

        "nda", "service_agreement", "consultancy_agreement",

        "cayman_islands", "english",

                              commercial_terms: dict, governing_law: str,

        Full contract generation pipeline:
        1. Validate contract_type and governing_law are supported

           knowledge/legal/jurisdictions/{governing_law}.md

        8. pdf_ops.create_pdf → PDF artifact (if output_format includes pdf)
        9. Return: { docx_path, pdf_path, key_terms_summary, word_count }
        """

                                 modifications: list, governing_law: str) -> dict:
        """
        Apply clause modifications to an existing draft.
        Returns: { updated_contract_text, change_log }
        """

    def get_contract_template(self, contract_type: str, jurisdiction: str) -> str:
        """

```markdown
# Singapore Business Law — Reference Guide

## Legal System
Common law jurisdiction. English law principles apply with Singapore statute modifications.
Primary courts: State Courts, High Court, Court of Appeal, Singapore International
Commercial Court (SICC).

## Key Statutes for Business

### Companies Act (Cap. 50)
- Governs incorporation, operation, and winding up of companies
- Private limited company (Pte. Ltd.) — most common structure
- Minimum 1 director ordinarily resident in Singapore
- Annual returns filed with ACRA
- AGM requirements (exemptions available for small private companies)

### Employment Act (Cap. 91)
- Covers employees earning ≤ SGD 4,500/month (salary threshold applies to
  Part IV provisions — salary caps adjusted periodically, verify current cap)
- Minimum notice periods: <26 weeks = 1 day; 26 weeks–2 years = 1 week;
  2–5 years = 2 weeks; >5 years = 4 weeks
- Annual leave: 7 days (first year), increasing by 1 day/year up to 14 days
- Sick leave: 14 days outpatient / 60 days hospitalisation
- CPF contributions required for Singapore citizens and PRs
- Mandatory retrenchment benefits: MOM guidelines (not statutory minimum)

### Personal Data Protection Act 2012 (PDPA)
- Governs collection, use, disclosure, and care of personal data
- Mandatory data breach notification within 3 business days if >500 individuals affected
  or likely to cause significant harm
- Financial penalties: up to SGD 1,000,000 or 10% of annual Singapore turnover
  (whichever higher, for organisations with annual turnover > SGD 10M)
- Data Protection Officer (DPO) appointment not mandatory but strongly recommended
- PDPC (Personal Data Protection Commission) is the regulator

### Arbitration Act (International) (Cap. 143A)
- Based on UNCITRAL Model Law
- Singapore is a leading arbitration seat globally
- SIAC Rules widely used; ICC, AAA-ICDR also used
- Singapore courts strongly pro-arbitration; very rare to set aside awards

### Intellectual Property
- Trade Marks Act (Cap. 332): registration at IPOS
- Patents Act (Cap. 221): 20-year protection; IPOS examination
- Copyright Act 2021: automatic protection, no registration required
- IP ownership: employer owns IP created in course of employment (default rule)

## Standard Contractual Practices
- Contracts enforceable in English; Chinese/Malay contracts valid but English preferred
- Electronic signatures: legally valid under Electronic Transactions Act
- Stamp duty payable on certain instruments (share transfers, property)
- Limitation period: 6 years for simple contracts; 12 years for contracts under seal

## Common Risk Areas for Foreign Companies
- Ensure at least 1 Singapore-resident director (nominee director services available)
- CPF obligations for local hires — non-compliance = criminal offence
- GST registration required if annual turnover > SGD 1,000,000
- Employment Pass required for foreign employees; quota and salary thresholds apply
- Non-compete clauses: enforceable but must be reasonable in scope, duration, geography
```
