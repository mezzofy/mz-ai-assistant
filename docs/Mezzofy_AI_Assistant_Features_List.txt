====================================================================
FILE STORAGE HIERARCHY
====================================================================

USER SCOPE (Personal Folder)
- Private files visible only to the user
- Default storage for individual work

DEPARTMENT SCOPE (Shared Department Folder)
- Visible to all members of the user's department
- Ideal for team collaboration

COMPANY SCOPE (Company-wide Public Folder)
- Visible to all staff across all departments
- Management-only write access
- Used for company-wide policies and announcements
